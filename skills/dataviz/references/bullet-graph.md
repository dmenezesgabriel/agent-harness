# Bullet Graph

**Use when**: showing a single key metric against a target and a qualitative performance range (e.g., poor / satisfactory / good) in a compact space — a space-efficient alternative to a gauge or speedometer chart.  
**Avoid when**: the audience is unfamiliar with the format and no legend space exists; you need to compare more than ~5–6 metrics at once (a table with conditional formatting may be cleaner).

## Key rules

- The primary bar (the "bullet") encodes the actual value
- The short perpendicular line encodes the target/goal
- The background bands encode qualitative ranges — use 3 neutral gray shades (light → dark) not color
- Label the metric name and unit clearly — the compact format collapses context
- Orient horizontally for readability; vertical bullet graphs are harder to scan

## Anti-patterns

- Using colors (red/yellow/green) for the background bands — adds emotional loading beyond the data; gray gradients are sufficient
- Gauge/speedometer charts as a substitute — they waste space and encode less information than a bullet graph
