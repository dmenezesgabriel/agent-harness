# Square Area Chart / Unit Chart / Waffle Chart

**Use when**: making abstract quantities tangible for a general audience — each unit represents a real entity (person, dollar, case, vote); the message is about proportions that feel more visceral when counted as discrete units.  
**Avoid when**: quantities are large (thousands+) — individual units become invisible dots; precise comparisons are more important than emotional resonance; the audience needs exact values.

## Key rules

- Each unit must represent the same quantity — inconsistent unit sizes destroy the encoding
- Keep unit count ≤ ~100 for square/waffle charts; use representative sampling (1 unit = N entities) for larger datasets and label the scale clearly
- Group units by category and use color to distinguish them
- Label the total count and the proportion (e.g., "32 of 100 patients") — the visual alone is imprecise
- Use in contexts where humanizing the data matters: public health, social impact, policy communication

## Anti-patterns

- Unit chart with inconsistent unit sizes or irregular grid — the comparison breaks down
- Using unit charts for precise analytical comparison — a bar or table is faster and more accurate
- Unlabeled sampling (showing 20 icons to represent 2,000 people without stating the scale) — misleads about absolute quantities
