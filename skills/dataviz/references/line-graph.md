# Line Graph

**Use when**: showing how a value changes over continuous time or sequence; comparing several trends simultaneously.  
**Avoid when**: data is categorical with no inherent relationship between points; dataset is sparse or discontinuous; you need to show total quantities (use bar instead).

## Key rules

- Max 4–5 series before switching to small multiples — readers can't track more simultaneously
- Zero baseline advisable but not mandatory; skip it when zero has no practical meaning on the scale or when the range is small relative to zero
- Data range should fill ~70–80% of vertical space — balanced whitespace above and below
- X-axis must represent a continuous or chronological variable — never connect unrelated categories with a line
- Direct-label lines at the end rather than using a legend

## Anti-patterns

- Line chart over categorical x-axis (e.g., product names) — implies false continuity; use bar instead
- More than 7 lines without small multiples — becomes "spaghetti"
- Connecting sparse, discontinuous data points — misleads about intermediate values
